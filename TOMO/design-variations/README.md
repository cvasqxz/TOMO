# TOMO - Design Variations

Esta carpeta contiene 30 diseños minimalistas diferentes para la página principal de TOMO.

## 🎨 Cómo Ver los Diseños

1. **Abrir el índice**:
   ```bash
   # Desde la carpeta design-variations
   open index.html
   # o en Linux
   xdg-open index.html
   ```

2. **Ver diseño individual**:
   ```bash
   open design-01-centered.html
   ```

## 🎯 Características

- ✅ **CSS embebido** - Cada archivo es independiente
- ✅ **Modulares** - Fáciles de copiar e integrar
- ✅ **Paleta consistente** - Dracula y Alucard
- ✅ **Fuentes originales** - Questrial y Kode Mono
- ✅ **Responsive** - Adaptables a diferentes pantallas
- ✅ **Funcionales** - Botones de "Open File" y "Theme Toggle"

## 🔧 Cómo Integrar un Diseño

1. **Elegir diseño favorito**
2. **Copiar estructura HTML** del archivo elegido
3. **Adaptar el CSS** a los estilos modulares de TOMO
4. **Integrar con el JavaScript** existente (eventos, file-loader, etc.)
