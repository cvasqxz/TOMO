# Nuevos 10 Diseños Agregados (21-30)

## 🎨 Diseños 21-30 - La Nueva Colección

### 21. Split Screen 🌓
**Concepto**: Pantalla dividida verticalmente
- **Tema**: Dual (Dark + Light simultáneamente)
- **Características**:
  - Lado izquierdo oscuro (Dracula)
  - Lado derecho claro (Alucard)
  - Contraste visual máximo
  - Perfecta simetría
- **Ideal para**: Mostrar ambos temas al mismo tiempo

### 22. Layers 📚
**Concepto**: Tarjetas apiladas con rotación
- **Tema**: Dark
- **Características**:
  - 3 capas apiladas rotadas
  - Efecto hover 3D
  - Animaciones suaves
  - Información por capa
- **Ideal para**: Presentación dinámica de contenido

### 23. Command Palette ⌨️
**Concepto**: Paleta de comandos estilo IDE
- **Tema**: Light
- **Características**:
  - Barra de búsqueda
  - Lista de comandos con iconos
  - Shortcuts de teclado visibles
  - Gradiente de header
- **Ideal para**: Power users y desarrolladores

### 24. Bento Grid 🍱
**Concepto**: Grid asimétrico japonés
- **Tema**: Dark
- **Características**:
  - Grid 4x3 irregular
  - Diferentes tamaños de celdas
  - Logo destacado con gradiente
  - Información modular
- **Ideal para**: Dashboard compacto

### 25. Floating 🎈
**Concepto**: Elementos flotantes animados
- **Tema**: Light
- **Características**:
  - Círculos de fondo animados
  - Logo con animación float
  - FABs (Floating Action Buttons)
  - Efecto glassmorphism ligero
- **Ideal para**: Diseño moderno y juguetón

### 26. Tabs 📑
**Concepto**: Interfaz con pestañas
- **Tema**: Dark
- **Características**:
  - Barra de tabs superior
  - Estilo VS Code/navegador
  - Indicador de tab activo
  - Área de contenido principal
- **Ideal para**: Múltiples archivos/documentos

### 27. Spotlight 💡
**Concepto**: Foco que sigue al cursor
- **Tema**: Dark
- **Características**:
  - Spotlight animado interactivo
  - Efecto glow en el título
  - Botones con ripple effect
  - JavaScript para seguir mouse
- **Ideal para**: Experiencia interactiva

### 28. Swiss Design 🇨🇭
**Concepto**: Grid riguroso suizo
- **Tema**: Light
- **Características**:
  - Grid 12x8 preciso
  - Tipografía bold y minimalista
  - Uso de espacio negativo
  - Acento rojo característico
- **Ideal para**: Puristas del diseño

### 29. Neumorphism 🔘
**Concepto**: Soft UI con sombras
- **Tema**: Light
- **Características**:
  - Sombras internas y externas
  - Botones "hundidos"
  - Efecto 3D sutil
  - Colores pastel
- **Ideal para**: Interfaces táctiles y modernas

### 30. Brutalist 🏗️
**Concepto**: Diseño brutal y directo
- **Tema**: Light/Dark mix
- **Características**:
  - Bordes gruesos (5-8px)
  - Tipografía oversized y bold
  - Contraste extremo
  - Sin gradientes ni sombras
  - All caps
- **Ideal para**: Hacer una declaración visual fuerte
